## Built-in sequences

### Container sequences:
* Can hold items of differente types, including nested containers ( list, tuple, collections.deque )
* Holds references to the objects it contains
### Flat sequences: 
* Hold items of one simple type ( str, bytes, array.array )
* Stores the value of its contents in its own memory space

### Mutable sequences
* list, bytearray, array.array, collections.deque

### Inmutable Sequences
* tuple, str, bytes
