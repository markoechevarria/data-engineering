# Python

Content about Data analytics, logging and apis

* ## [Fluent Python](fluent-python/)

    * The Data Model: How to use special methods (or "dunder" methods) to make your custom objects behave exactly like built-in Python types.
    * Data Structures: Mastering sequences, dictionaries, sets, and the complexities of handling Unicode (text vs. bytes).
    * Functions as Objects: Treating functions as first-class objects to implement classic design patterns cleanly, alongside decorators and closures.
    * Control Flow: Building custom iterables, context managers, generators, and asynchronous code (like async / await).
    * Metaprogramming: Exploring properties, attribute descriptors, class decorators, and metaprogramming hooks to create highly dynamic and reusable frameworks.
