# Git and Github

## What are Git and Github

* Git is a command line program which allows you to track versions of any code or plain text documents that you create
* GitHub is a website that provides remote Git repositories. A remote repository is just a Git repository that you’re able to access over an internet connection.

## Setting up Git and Github

* It is needed to set up a Git user name, and a email address 

```
git --version
git config --global user.name "myUserName"
git config --global user.email myName@email.com
```

## Getting started with Git

* To start tracking files with Git in a directory `git init`
* Get information about the status of the git repository `git status`
* To let git know what we want to track a file we need to use `git add`


```
git init
git status
git add file.txt
```
