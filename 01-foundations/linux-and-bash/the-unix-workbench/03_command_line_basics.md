# Command line basics

* Bash Shell commands tend to have the following structue 
```
[comamnd] [options] [arguments]
```

### Commands

* clear: clean up the terminal
* echo : prints a phrase to the console
* pwd: prints the working directory
* cd: change the working directory
* ls: list files and folders in a directory
* mkdir: create a directory
* touch: create a blank file
* wc: display the number of lines, number of words, and number of characters
* cat: print a file to the console
* less: allow view multi-page files
* head: glimpse the beginning of a file
* tail: glimpse the end of a file
* \> : redirect the output of a command into a file
* \>\> : append command output to the enf of a file
* mv: move or rename files or folders
* cp: copy files or folders
* rm: remove files or folders
