# Data warehousing, Businesss Inteligence and Dimensional Modeling

## Different wordls of data captura and data analysis

Information: one of the most important assets of any organizations. It is almost always used for Operational record keeping (where the data is stored) and Analytical decision making ( where the data is extracted from )

Operational systems process transactions quickly, almost always del with one transaction record at a time. They reflect the most current state.

DW/BI systems are optimized for high-performace queries. Demands the historical context be preserver to accurately evaluate over the time.

## Goals of Data warehousing and Business Inteligence

* The DW/BI system must make information easily accesible. The data's structures and labels should mimic the business users' thought processes and vocabulary. Simple and fast.
* The DW/BI system must present information consistently. The data must be credible. Common labels and definitions are used across data sources.
* The DW/BI system must adapt to change. The system must be designed to handle the inevitable change gracefully do that it doesnt invalidate existing data or applications.
* The DW/BI system must present information in a timely way. 
* The DW/BI system must be a secure bastion that protects the information assets.
* The DW/BI system must serve as the authoritative and trustworthy foundation for improved decision making. The DW must have the right data to support decision making.
* The business comunity must accept the DW/BI system to deem it sucessful.
