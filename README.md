Data Engineering learning repository! This repo serves as a comprehensive log of my progress, notes, code snippets, and projects as I follow a structured Data Engineering roadmap.

- [ ] [**SQL**](): Window functions, CTEs, Query optimization. `Book: SQL Performance Explained`

- [ ] [**Data Structures and Algorithms**](): Hash maps, heaps, trees, sorting, searching, complexity, streaming algorithms. `Book: Cracking the Coding Interview`

- [ ] [**OLAP**](): Star Schema, Snowflake Schema, Redshift, Snowflake, BigQuery, Data Warehousing concepts, Partitioning & clustering. `Book: The Data Warehouse Toolkit`

- [ ] [**Data Lake**](): Data Lake / Lakehouse Architecture. `Book: Fundamentals of Data Engineering`

- [ ] [**dbt (Data Build Tool)**](): Models, Tests, Macros, Incremental models, Data lineage. `Book: Unlocking dbt`

- [ ] [**Streaming Processing**](): Kafka, Producers/consumers, Topics & partitions, Offsets, Exactly-once semantics. `Book: Kafka: The Definitive Guide`

- [ ] [**Distributed System Fundamentals**](): How data is partitioned and processed with clusters. `Book: Designing Data-Intensive Applications`

### Phase 6: Extra / Advanced Topics
